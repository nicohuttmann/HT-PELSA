
# HT-PELSA 

This repository contains

1. the R project to reproduce the data and figures in the HT-PELSA study `Code_to_reproduce_the_figures`, 
2. the `HT.PELSA` R package to analyze and visualize your own PELSA data in the branch [R-package](https://github.com/nicohuttmann/HT-PELSA/tree/R-package), 
3. the accompanying descriptions of PELSA data analysis and the package [here]() and 
4. the source data used in the [HT-PELSA Shiny app](https://apps.embl.de/htpelsaapp/) in `Shiny_app_source_data`.

## R package

To install the `HT.PELSA` R package, use the following code: 

```
remotes::install_github("nicohuttmann/HT-PELSA@R-package")
```
