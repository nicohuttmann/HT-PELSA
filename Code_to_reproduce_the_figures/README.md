# HT-PELSA

This repository contains the code used for the data analysis of the manuscript "High-throughput peptide-centric local stability assay extends protein-ligand identification to membrane proteins, tissues, and bacteria". Required packages are listed in "packages.R".

**Please kindly note that the Fig.S here correspond to the Extended Data Figs in the manuscript.** 



## 1\_Benchmarking\_Fig.1B and S1

This directory provides all the input datasets, fasta, known database, and code for the benchmarking experiments related to Fig 1B and S1:

* The input datasets, obtained from DIA-NN (V 1.8.1), correspond to files ending with "pr\_matrix.tsv"
* The human fasta file, obtained from Uniprot, correspond to the file ending with ".fasta"
* Contamination protein list, obtained from Maxquant, correspond to the file "contamination\_protein.csv"
* Kinase database, obtained from Kinhub, correspond to the file "kinase\_hub.csv"
* Kinase domain database, obtained from Uniprot, correspond to the file "kinase\_domain\_hub.csv"

The other files are intermediate outputs generated by the code (.Rmd file) and are necessary for the figures.
